Для тестов на локальной машине с нуля:

Крайне рекомендую работать для каждого проекта в своей виртуальной среде, иначе, если устанавливать все необходимые для каждого проекта библиотеки в одну, однажды всё крашнется и восстановить будет очень сложно/невозможно.

Гуглится по запросу python virtual envs.
Удобнее всего работать с ними через Anaconda, но это ваше дело, можно и без неё.

1. Создайте виртуальную среду, если используете Anaconda, то в Anaconda Prompt вводите:
	conda create -n img_comp pip python=3.6
2. Активируйте её
	conda activate img_comp
3. Перейдите в эту директорию
	cd path, где path - ваша папка, просто скопируйте путь из проводника
4. Для установки необходимых библиотек выполните:
	pip install -r requirements.txt
5. Всё готово! Теперь можно выполнить:
	python test_diff.py
